
# heroku-pizza

# Welcome to our Pizza Store

## Demo

For the frontend, I used React, Redux, for backend - Node JS, Express, for DB - Mongo. 

https://heroku-pizza.herokuapp.com/

### To run locally

1. Frontend :
   npm i,
   npm start
  
2. Backend: 
   npm i,
   npm start --port 3001
3. Seed DB :
   npm run seed;
   
### connection string to MongoDB can be provided by request
